✅ 변환이 끝났습니다. 아래 파일을 다운로드해서 엑셀로 열면 바로 인쇄할 수 있어요. 갑지는 가로 방향, 한 페이지 맞춤으로 설정되어 있습니다.

1. 시트 합산 — "오전" 계열 시트(오전, 오전 (2)…)와 "오후" 계열 시트를 각각 하나로 모읍니다.

2. 정렬 기준
신청업체 1차: 최형재팀 → 박인기팀 → 유성완팀 → 연호 → 하나기술 → 엘테크 → 엘테크(SGAS) → 연호(사외) → 하나기술(사외)
구분 2차: GAS → PCW → 구조관 → 코팅배관 → SPIRAL → PVC → 서포트
목록에 없는 업체·구분은 맨 뒤로 정렬됩니다.

3. 시트 분할 — 정렬된 목록을 10건 단위로 새 시트(오전, 오전 (2), …)에 다시 채웁니다.

4. 갑지 집계 — 업체 × 구분별 포장개수 합계표를 만듭니다. "엘테크"와 "엘테크(SGAS)"는 갑지에서 하나로 합산됩니다. 목록 밖의 항목은 "기타" 행/열로 집계됩니다.

원본 파일과 서식(글꼴·테두리·배경색)은 최대한 보존되며,
숫자·셀 구조 위주로 만들어졌습니다. 변환 결과는 항상 다운로드 전 미리 확인해 주세요.
